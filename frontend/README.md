# 🍳 Recetas Deliciosas - Frontend

Frontend del proyecto "Recetas Deliciosas Como Servicio" para la materia de Temas Selectos de Criptografía.

## 📋 Descripción

Aplicación web que permite a una chef compartir recetas cifradas con sus suscriptores mediante criptografía híbrida.

## 🚀 Tecnologías

- React.js con Vite
- Tailwind CSS
- React Router DOM
- Web Crypto API (para criptografía)
